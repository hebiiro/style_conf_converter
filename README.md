# 🎉スタイルファイル変換

* このアプリはaviutl2の`style.conf`を別の形式に変換します。

## 💡使い方

1. `入力`に`style.conf`ファイルのパスを指定します。
1. `出力`に出力用ファイルのパスを指定します。
	* 空の場合は`入力`のフォルダに`style.yml`が作成されます。
1. `変換`ボタンを押します。
